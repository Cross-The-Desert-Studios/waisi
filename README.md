# What Am I Stepping In? (WAISI)

What Am I Stepping In? is a mod for Minecraft that tells the player what they are standing on, or stepping in! Inspired by Waila/Hwyla/Jade.

## Features

- **HUD Display:** Shows the name of the block directly under your feet (or the one you are standing in, like water or tall grass).
- **Mod Name:** Identifies which mod the block belongs to.
- **Customization:**
  - **Interactive Positioning:** Drag and drop the HUD to place it anywhere on your screen.
  - **Custom Colors:** Configure background, border, and text colors using hex codes.
  - **Styles:** Adjust scale, transparency, and rounded corners to match your UI preference.
- **Mod Menu Integration:** Easily access settings via the Mod Menu config screen.
